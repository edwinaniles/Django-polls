Creating a Poll (Admin Side)
Open your browser and go to the admin page:
/admin/
Log in using your admin (superuser) credentials.
On the dashboard, click “Questions”.
Click “Add Question”.
Enter:
The poll question (e.g., “What is your favorite food?”)
The publication date and time
Save the question.
Add choices to the poll:
Open the question you just created
Add options (e.g., Pizza, Burger, Salad)
You can add multiple choices
Save everything.
Voting in a Poll (User Side)
Go to the polls page:
/polls/
You will see a list of available polls.
Click on any poll question.
Select one of the available choices.
Submit your vote.
Viewing Results
After voting, you will be redirected to the results page.
You can see:
Total votes for each choice
Which option is most popular
